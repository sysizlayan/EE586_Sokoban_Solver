1) If you do not have python 3.6, please install anaconda installation package.
2) After installation, run install.bat as administrator for once.

3) Run run.bat to solve sokoban puzzle online from web-based GUI.

4) If you want text based GUI, run the ee586.exe with text-fortmatted map and wanted algorithm.
e: BFS
t: IDDFS
y: Deadlock
u: Manhattan
i: Euclidean
o: Manhattan and Deadlock
p: Deadlock Checking BFS

example input:

ee586.exe sokobanGame.txt o 
--> Solve the problem in sokobanGame.txt using Deadlock and Manhattan Heuristic
